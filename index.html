Fable-graphics

Explore Fable-graphics through demos!

Fable-graphics is a project grouping several binding for graphics library with Fable.

The current bindings available are:

  • dc.js
  • d3js
  • PixiJS (only v3 for the moment)
  • Three.js

You can also contribute more examples by sending us a pull request for the samples directory!

PixiJS

Here is a selection of some official Pixi.js sampes ported to F#/Fable. They work with Pixi.js v3.0.11 and v4.x.x although we don't support v4 officially at the moment.

Each sample displays both the resulting animation and the F# source code

  • Basic - Your very first pixi sample
  • Graphics - How to use the graphics API
  • Dragging - Play with mouse events and add interactivity
  • Blur Filter - How to use a common filter
  • Text - Add simple text and bitmap text with style!
  • MovieClip - Load your spritesheet in a movieclip fashion
  • Particles - Zounds of sprites with ParticleContainer!
  • Render Texture - Eye bleeding effects!
  • Masking - Masking unleashed!
